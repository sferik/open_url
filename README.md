# open_url

A cross-platform Crystal library for opening a URL in a browser.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     open_url:
       github: sferik/open_url
   ```

2. Run `shards install`

## Usage

```crystal
require "open_url"

OpenURL.open("http://example.com")
```
